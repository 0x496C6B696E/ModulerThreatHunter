# Modüler Tehdit Avcısı

Amaç: Multi-language (JS, Python, Java, Bash/PowerShell, SQL, Go, Rust, Ruby, Assembly) öğrenme-odaklı, modüler siber güvenlik lab'ı.

Başlangıç:
1. Repo klonla / oluştur.
2. `coordinator` ile modülleri listele/çalıştır.
3. `docker-compose` (isteğe bağlı labs) ile test ortamı ayağa kaldır.

Güvenlik: Tüm testleri **yalnızca** kendi lokal/izole VM/container ortamında yap.
